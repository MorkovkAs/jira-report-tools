# report-service-ui

It is a Vue client for getting Jira issues, getting reports on them, getting release notes.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
